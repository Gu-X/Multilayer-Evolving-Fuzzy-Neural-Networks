# Multilayer-Evolving-Fuzzy-Neural-Networks
MATLAB code for the research paper titled "Multilayer Evolving Fuzzy Neural Networks".
This work is described in:
===============================================================================================================================
Gu, X., Angelov, P., Han, J., & Shen, Q. (2023). Multilayer Evolving Fuzzy Neural Networks. 
IEEE Transactions on Fuzzy Systems, vol.31, no.12, pp. 4158-4169.
===============================================================================================================================
Please cite the paper above if this code helps.
